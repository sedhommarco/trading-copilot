# Trading Copilot

A minimal SPA for surfacing current trading opportunities and strategy-level context.

> ⚠️ Trading involves substantial risk of loss. This tool does not provide financial advice.

---

## Live app

Deployed via GitHub Pages — see the [Actions tab](https://github.com/sedhommarco/trading-copilot/actions) for deployment status.

---

## Run locally

No build step required (current vanilla SPA):

```bash
git clone https://github.com/sedhommarco/trading-copilot.git
cd trading-copilot
# Open app/index.html in your browser, or serve with:
npx serve app
```
